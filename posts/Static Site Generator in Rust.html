rusty & rustful

This site is generated with rustful which is a rewrite of rusty but with a different approach to templating & general project structure. It's still a work in progress but it's already usable.

Technical Details

Rustful is basically identical to rusty, except rustful uses tera for templating instead of rusty's maud which is a compile-time html templating library. This means that rustful is much slower than rusty, but it's also much more flexible. It also means that rustful is much more similar to other static site generators like jekyll or hugo.

Usage

$ rustful [options]

Options

  • -h, --help - Show this help message
  • -v, --version - Show the version
  • -s, --source - The source directory
  • -o, --output - The output directory

Example

$ rustful -s source -o output

Config

The config file is located at source/config.toml and has the following structure:

title = "My Blog"
description = "A blog about stuff"

Posts

Posts are located in source/posts and have the following structure:

---
title: My Post
date: 2014-01-01
author: John Doe
---

# My Post

Templates

Templates are located in source/templates and use Tera (jinja2-like) syntax.

Assets

Assets are located in source/assets and are copied to the output directory.
